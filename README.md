## Trading Bots

- [Freqtrade](https://github.com/freqtrade/freqtrade) - Free, open source crypto trading bot written in Python.
- [Hummingbot](https://github.com/hummingbot/hummingbot) - Open source market making and arbitrage bot.
- [Jesse](https://github.com/jesse-ai/jesse) - Advanced crypto trading framework for Python.
- [Gekko](https://github.com/askmike/gekko) - Bitcoin trading bot with backtesting and paper trading.
- [Zenbot](https://github.com/DeviaVir/zenbot) - Lightweight, extendable crypto trading bot.
- [OctoBot](https://github.com/Drakkar-Software/OctoBot) - Open source trading bot with AI capabilities.
- [NOFX AI Trading OS](https://github.com/pjl914335852-ux/ai-deployment-service) - AI-powered automated trading system with multi-exchange and multi-AI support.

## AI Models for Trading

- [FinGPT](https://github.com/AI4Finance-Foundation/FinGPT) - Open-source financial large language model.
- [FinRL](https://github.com/AI4Finance-Foundation/FinRL) - Deep reinforcement learning for quantitative finance.
- [Stock-Prediction-Models](https://github.com/huseinzol05/Stock-Prediction-Models) - Collection of ML models for stock prediction.
- [TradingGPT](https://github.com/microsoft/TradingGPT) - Multi-agent framework for financial trading.
- [ChatGPT/Claude for Trading](https://github.com/pjl914335852-ux/ai-deployment-service#-nofx-ai-trading-os) - Using LLMs for trade signal generation and strategy optimization.

## Strategy Frameworks

- [Backtrader](https://github.com/mementum/backtrader) - Python backtesting library for trading strategies.
- [Zipline](https://github.com/quantopian/zipline) - Pythonic algorithmic trading library.
- [QuantConnect](https://github.com/QuantConnect/Lean) - Open-source algorithmic trading engine.
- [VectorBT](https://github.com/polakowo/vectorbt) - Backtesting at scale with vectorized operations.

## Data & Analytics

- [CCXT](https://github.com/ccxt/ccxt) - Unified crypto exchange API library (100+ exchanges).
- [Chartscout](https://chartscout.io) - Real time crypto Detects chart patterns instantly and alerts you in under 20 seconds.
- [CryptoFeed](https://github.com/bmoscon/cryptofeed) - Cryptocurrency exchange websocket data feed handler.
- [TradingView](https://www.tradingview.com/) - Charting platform with social trading features.
- [CoinGecko API](https://www.coingecko.com/en/api) - Free crypto market data API.
- [Messari](https://messari.io/api) - Crypto data and research API.
